рЃърЃўрЃарЃљрЃЊрЃў рЃюрЃЮрЃЏрЃћрЃарЃў рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА рЃАрЃљрЃЦрЃљрЃарЃЌрЃЋрЃћрЃџрЃЮрЃА рЃЏрЃЮрЃЦрЃљрЃџрЃљрЃЦрЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃа рЃАрЃљрЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃЮ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃърЃўрЃарЃА рЃћрЃюрЃўрЃГрЃћрЃЉрЃљ рЃЊрЃљрЃЉрЃљрЃЊрЃћрЃЉрЃўрЃА рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃерЃў. рЃљрЃдрЃюрЃўрЃерЃюрЃБрЃџрЃў рЃюрЃЮрЃЏрЃћрЃарЃў рЃЊрЃљрЃбрЃљрЃюрЃўрЃџрЃўрЃљ рЃЏрЃЮрЃЦрЃљрЃџрЃљрЃЦрЃўрЃА рЃЮрЃцрЃўрЃфрЃўрЃљрЃџрЃБрЃа рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃќрЃћ(рЃърЃўрЃарЃљрЃЊрЃЮрЃЉрЃўрЃА рЃЏрЃЮрЃгрЃЏрЃЮрЃЉрЃљ, рЃърЃљрЃАрЃърЃЮрЃарЃбрЃў, рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃЏрЃЮрЃгрЃЏрЃЮрЃЉрЃљ рЃЊрЃљ рЃАрЃ«рЃЋрЃљ). рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџ рЃЏрЃЮрЃЦрЃљрЃџрЃљрЃЦрЃћрЃА рЃљрЃЦрЃЋрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌрЃў рЃърЃўрЃарЃљрЃЊрЃў рЃюрЃЮрЃЏрЃћрЃарЃў, рЃ«рЃЮрЃџрЃЮ рЃћрЃарЃЌрЃў рЃЊрЃљ рЃўрЃЏрЃљрЃЋрЃћ рЃюрЃЮрЃЏрЃарЃўрЃА рЃЏрЃўрЃюрЃўрЃГрЃћрЃЉрЃљ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃърЃўрЃарЃўрЃАрЃЌрЃЋрЃўрЃА рЃЊрЃљрЃБрЃерЃЋрЃћрЃЉрЃћрЃџрЃўрЃљ. рЃљрЃАрЃћрЃЋрЃћ рЃЊрЃљрЃБрЃерЃЋрЃћрЃЉрЃћрЃџрЃўрЃљ рЃБрЃЎрЃЋрЃћ рЃЏрЃўрЃюрЃўрЃГрЃћрЃЉрЃБрЃџрЃў рЃюрЃЮрЃЏрЃарЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ рЃљрЃю рЃњрЃљрЃБрЃЦрЃЏрЃћрЃЉрЃљ, рЃњрЃљрЃарЃЊрЃљ рЃЎрЃљрЃюрЃЮрЃюрЃўрЃЌ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃБрЃџрЃў рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃБрЃџрЃў рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃћрЃЉрЃўрЃАрЃљ. рЃърЃўрЃарЃљрЃЊрЃў рЃюрЃЮрЃЏрЃарЃўрЃА рЃЏрЃўрЃюрЃўрЃГрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў рЃЊрЃљ рЃДрЃЋрЃћрЃџрЃљ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃў рЃљрЃдрЃўрЃарЃўрЃфрЃ«рЃћрЃЉрЃљ рЃАрЃљрЃљрЃњрЃћрЃюрЃбрЃЮрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљрЃерЃў.

рЃърЃўрЃарЃљрЃЊрЃў рЃюрЃЮрЃЏрЃћрЃарЃў рЃерЃћрЃЊрЃњрЃћрЃЉрЃљ рЃЌрЃћрЃарЃЌрЃЏрЃћрЃбрЃў рЃфрЃўрЃцрЃарЃўрЃАрЃљрЃњрЃљрЃю. рЃЏрЃўрЃАрЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА рЃљрЃЊрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃБрЃџрЃў рЃћрЃарЃЌрЃћрЃБрЃџрЃўрЃА рЃЎрЃЮрЃЊрЃА, рЃАрЃљрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃЮ рЃфрЃўрЃцрЃарЃА, рЃАрЃљрЃљрЃњрЃћрЃюрЃбрЃЮрЃА рЃбрЃћрЃарЃўрЃбрЃЮрЃарЃўрЃБрЃџрЃў рЃАрЃљрЃЏрЃАрЃљрЃ«рЃБрЃарЃўрЃА рЃЎрЃЮрЃЊрЃА рЃЊрЃљ рЃарЃўрЃњрЃўрЃЌ рЃюрЃЮрЃЏрЃћрЃарЃА. рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА рЃАрЃърЃћрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃерЃћрЃќрЃдрЃБрЃЊрЃЋрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃЌрЃљрЃф рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃБрЃџрЃўрЃљ рЃЊрЃљрЃБрЃерЃЋрЃћрЃЉрЃћрЃџрЃў рЃфрЃўрЃцрЃарЃБрЃџрЃў рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃљрЃфрЃўрЃћрЃЉрЃў. рЃАрЃљрЃќрЃдрЃЋрЃљрЃарЃњрЃљрЃарЃћрЃЌ рЃЏрЃДрЃЮрЃцрЃў рЃАрЃљрЃЦрЃљрЃарЃЌрЃЋрЃћрЃџрЃЮрЃА рЃЏрЃЮрЃЦрЃљрЃџрЃљрЃЦрЃћрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃърЃўрЃарЃљрЃЊрЃў рЃюрЃЮрЃЏрЃарЃўрЃА рЃърЃўрЃарЃЋрЃћрЃџ рЃЮрЃа рЃфрЃўрЃцрЃарЃА рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА рЃАрЃљрЃњрЃљрЃарЃћрЃЮ рЃАрЃљрЃЦрЃЏрЃћрЃЌрЃљ рЃАрЃљрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃЮрЃА рЃЎрЃЮрЃЊрЃў, рЃ«рЃЮрЃџрЃЮ рЃАрЃ«рЃЋрЃљ рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃў рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃћрЃЉрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА рЃЊрЃўрЃърЃџрЃЮрЃЏрЃљрЃбрЃўрЃБрЃа рЃгрЃљрЃарЃЏрЃЮрЃЏрЃљрЃЊрЃњрЃћрЃюрЃџрЃЮрЃЉрЃљрЃАрЃљ рЃљрЃю рЃАрЃљрЃЎрЃЮрЃюрЃАрЃБрЃџрЃЮ рЃЊрЃљрЃгрЃћрЃАрЃћрЃЉрЃБрЃџрЃћрЃЉрЃљрЃќрЃћ.

рЃљрЃЏрЃњрЃЋрЃљрЃарЃљрЃЊ, рЃърЃўрЃарЃљрЃЊрЃў рЃюрЃЮрЃЏрЃћрЃарЃў рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА рЃЏрЃЮрЃЦрЃљрЃџрЃљрЃЦрЃўрЃА рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃфрЃўрЃарЃћрЃЉрЃўрЃА рЃЏрЃЌрЃљрЃЋрЃљрЃа рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљрЃерЃў, рЃАрЃљрЃљрЃарЃЦрЃўрЃЋрЃЮ рЃЏрЃљрЃАрЃљрЃџрЃћрЃЉрЃАрЃљ рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃбрЃўрЃърЃўрЃА рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљрЃерЃў рЃърЃўрЃарЃЮрЃЋрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃЏрЃЮрЃФрЃўрЃћрЃЉрЃўрЃА рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЮрЃърЃбрЃўрЃЏрЃљрЃџрЃБрЃа рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА.

рЃАрЃљрЃЏрЃћрЃфрЃюрЃўрЃћрЃарЃЮ рЃЏрЃўрЃќрЃюрЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃАрЃљрЃГрЃўрЃарЃЮ рЃњрЃљрЃ«рЃЊрЃћрЃА, рЃћрЃарЃЌрЃў рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃљ рЃЊрЃљ рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃЌрЃљрЃю рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃћрЃџрЃћрЃЦрЃбрЃарЃЮрЃюрЃБрЃџ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљрЃерЃў. рЃљрЃЏрЃљрЃЋрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ, рЃљрЃдрЃюрЃўрЃерЃюрЃБрЃџрЃў рЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ рЃљрЃа рЃБрЃюрЃЊрЃљ рЃБрЃЦрЃЏрЃюрЃўрЃЊрЃћрЃА рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃА рЃАрЃљрЃцрЃарЃЌрЃ«рЃћрЃА рЃЊрЃљ рЃљрЃа рЃЦрЃЏрЃюрЃўрЃЊрЃћрЃА рЃЎрЃЮрЃюрЃцрЃўрЃЊрЃћрЃюрЃфрЃўрЃљрЃџрЃБрЃарЃЮрЃЉрЃўрЃА рЃЊрЃљрЃарЃдрЃЋрЃћрЃЋрЃўрЃА рЃарЃўрЃАрЃЎрЃА.

рЃљрЃдрЃюрЃўрЃерЃюрЃБрЃџрЃў рЃљрЃЏрЃЮрЃфрЃљрЃюрЃўрЃА рЃњрЃљрЃЊрЃљрЃАрЃљрЃГрЃарЃћрЃџрЃљрЃЊ, рЃцрЃљрЃарЃЌрЃЮрЃЊ рЃљрЃарЃўрЃА рЃЊрЃљрЃюрЃћрЃарЃњрЃўрЃџрЃў рЃЎрЃарЃўрЃърЃбрЃЮрЃњрЃарЃљрЃцрЃўрЃБрЃџрЃў рЃ░рЃћрЃерЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃЌрЃљрЃю рЃЊрЃљрЃАрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃџрЃљрЃЊ [1, 2]. рЃўрЃЊрЃћрЃљ рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃА рЃўрЃЏрЃљрЃерЃў, рЃарЃЮрЃЏ рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃўрЃА рЃЏрЃљрЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃфрЃўрЃарЃћрЃЉрЃћрЃџрЃў рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃЏрЃфрЃЋрЃћрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃў (рЃАрЃљрЃЦрЃљрЃарЃЌрЃЋрЃћрЃџрЃЮрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў - рЃърЃўрЃарЃљрЃЊрЃў рЃюрЃЮрЃЏрЃћрЃарЃў) рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃЊрЃћрЃА рЃАрЃърЃћрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃўрЃЌ рЃЊрЃљ рЃњрЃљрЃарЃЊрЃљрЃўрЃЦрЃЏрЃюрЃљрЃА рЃўрЃњрЃЋрЃљрЃа рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃљрЃЊ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЊрЃљрЃарЃЕрЃћрЃЉрЃљ рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃЊрЃљ рЃљрЃЏрЃљрЃЋрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃљрЃЊ рЃљрЃюрЃЮрЃюрЃўрЃЏрЃБрЃарЃў, рЃЏрЃўрЃАрЃў рЃБрЃЎрЃБ-рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљ рЃЊрЃљ рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃўрЃА рЃЏрЃљрЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃфрЃўрЃарЃћрЃЉрЃћрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃерЃћрЃБрЃФрЃџрЃћрЃЉрЃћрЃџрЃў [3, 4, 5].

Python-рЃўрЃА рЃАрЃЎрЃарЃўрЃърЃбрЃў, рЃЊрЃљрЃАрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃћрЃџ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљрЃерЃў рЃърЃЮрЃБрЃџрЃЮрЃЉрЃА рЃЌрЃћрЃарЃЌрЃЏрЃћрЃюрЃб-рЃюрЃўрЃерЃюрЃљ рЃърЃўрЃарЃљрЃЊ рЃюрЃЮрЃЏрЃћрЃарЃА, рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃўрЃА рЃЏрЃљрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ рЃЊрЃљ рЃърЃўрЃарЃљрЃЊ рЃюрЃЮрЃЏрЃћрЃарЃА рЃљрЃюрЃљрЃфрЃЋрЃџрЃћрЃЉрЃА рЃЏрЃўрЃАрЃў рЃ░рЃћрЃерЃўрЃЌ (рЃћ.рЃг. "рЃљрЃюрЃЮрЃюрЃўрЃЏрЃБрЃарЃў рЃЎрЃЮрЃЊрЃўрЃЌ"). рЃАрЃЎрЃарЃўрЃърЃбрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА SHA3-512 рЃЎрЃарЃўрЃърЃбрЃЮрЃњрЃарЃљрЃцрЃўрЃБрЃџрЃў рЃ░рЃћрЃерЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃА рЃћрЃбрЃљрЃърЃќрЃћ рЃўрЃЌрЃЋрЃџрЃћрЃЉрЃљ рЃЏрЃЮрЃгрЃўрЃюрЃљрЃЋрЃћ рЃЋрЃћрЃарЃАрЃўрЃљрЃЊ [6].

рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃърЃўрЃарЃљрЃЊрЃў рЃюрЃЮрЃЏрЃћрЃарЃў рЃЊрЃљрЃљрЃЏрЃБрЃерЃљрЃЮрЃА рЃЮрЃарЃњрЃЋрЃљрЃарЃљрЃЊ. рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў - рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃ░рЃћрЃерЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃърЃўрЃарЃљрЃЊрЃў рЃюрЃЮрЃЏрЃарЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљрЃА. рЃљрЃАрЃћрЃЌ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃерЃћрЃБрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЏрЃўрЃдрЃћрЃЉрЃБрЃџрЃў рЃ░рЃћрЃерЃўрЃА рЃБрЃЎрЃБ-рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљ рЃърЃўрЃарЃљрЃЊ рЃюрЃЮрЃЏрЃарЃљрЃЊ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЌрЃБ рЃ░рЃћрЃерЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃцрЃџрЃЮрЃЉрЃћрЃџрЃўрЃА рЃ«рЃћрЃџрЃерЃў рЃљрЃдрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ рЃърЃўрЃарЃљрЃЊрЃў рЃюрЃЮрЃЏрЃћрЃарЃў, рЃЏрЃљрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃўрЃњрЃўрЃЋрЃћ рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃўрЃЌ рЃЊрЃљрЃљрЃЏрЃБрЃерЃљрЃЮрЃА рЃўрЃА рЃЊрЃљ рЃЏрЃўрЃдрЃћрЃЉрЃБрЃџрЃў рЃ░рЃћрЃерЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃЏрЃЮрЃФрЃћрЃЉрЃюрЃЮрЃА рЃБрЃЎрЃЋрЃћ рЃљрЃарЃАрЃћрЃЉрЃБрЃџ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљрЃерЃў. рЃљрЃЏрЃњрЃЋрЃљрЃарЃљрЃЊ рЃЊрЃљрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃЮрЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃърЃўрЃарЃљрЃЊрЃў рЃюрЃЮрЃЏрЃарЃўрЃА рЃЏрЃцрЃџрЃЮрЃћрЃџрЃў рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ. рЃљрЃдрЃюрЃўрЃерЃюрЃБрЃџрЃў рЃарЃўрЃАрЃЎрЃўрЃА рЃљрЃдрЃЏрЃЮрЃАрЃљрЃцрЃ«рЃЋрЃарЃћрЃџрЃљрЃЊ, рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃўрЃўрЃА рЃЏрЃћрЃЮрЃарЃћ рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ - рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃърЃўрЃарЃљрЃЊ рЃюрЃЮрЃЏрЃћрЃарЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃћ.рЃг. рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЊрЃљ рЃ░рЃћрЃерЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃърЃўрЃарЃљрЃЊрЃў рЃюрЃЮрЃЏрЃарЃўрЃАрЃљ рЃЊрЃљ рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃљрЃфрЃўрЃўрЃЌ рЃерЃћрЃЊрЃњрЃћрЃюрЃўрЃџрЃў.

рЃарЃЮрЃњрЃЮрЃарЃф рЃърЃўрЃарЃљрЃЊрЃў рЃюрЃЮрЃЏрЃћрЃарЃў, рЃљрЃАрЃћрЃЋрЃћ рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЏрЃцрЃџрЃЮрЃЉрЃћрЃџрЃЌрЃљрЃю, рЃ«рЃЮрЃџрЃЮ рЃЏрЃЎрЃЋрЃџрЃћрЃЋрЃљрЃарЃА рЃњрЃљрЃЊрЃљрЃћрЃфрЃћрЃЏрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљ рЃАрЃљрЃЊрЃљрЃф рЃърЃўрЃарЃЮрЃЋрЃюрЃћрЃЉрЃўрЃА рЃАрЃљрЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃЮрЃЊ рЃЕрЃљрЃАрЃЏрЃБрЃџрЃўрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃ░рЃћрЃерЃў. рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃљрЃюрЃљрЃџрЃўрЃќрЃљрЃЏрЃЊрЃћ рЃЏрЃўрЃАрЃў рЃљрЃЏрЃњрЃЋрЃљрЃарЃў рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃА рЃ«рЃЊрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃЊрЃћрЃА рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃЊ.

**рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃљ:** рЃарЃљрЃЊрЃњрЃљрЃю рЃАрЃљрЃЦрЃљрЃарЃЌрЃЋрЃћрЃџрЃЮрЃерЃў рЃ»рЃћрЃа рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА рЃЊрЃљрЃЏрЃЎрЃЋрЃўрЃЊрЃарЃћрЃЉрЃБрЃџрЃў рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃў, рЃЌрЃБ рЃарЃЮрЃњрЃЮрЃа рЃБрЃюрЃЊрЃљ рЃЏрЃЮрЃ«рЃЊрЃћрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃћрЃЉрЃерЃў рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃЌрЃљрЃю рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃљ, рЃљрЃдрЃюрЃўрЃерЃюрЃБрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃерЃћрЃЋрЃЦрЃЏрЃћрЃюрЃў рЃЕрЃћрЃЏрЃў рЃАрЃљрЃЊрЃЮрЃЦрЃбрЃЮрЃарЃЮ рЃЊрЃўрЃАрЃћрЃарЃбрЃљрЃфрЃўрЃўрЃА рЃЎрЃЋрЃџрЃћрЃЋрЃљрЃерЃў рЃњрЃљрЃЏрЃЮрЃАрЃљрЃДрЃћрЃюрЃћрЃЉрЃџрЃљрЃЊ. рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ, рЃЊрЃљрЃћрЃ«рЃЏрЃљрЃарЃЮрЃА рЃЏрЃЎрЃЋрЃџрЃћрЃЋрЃљрЃарЃћрЃЉрЃА рЃЏрЃАрЃњрЃљрЃЋрЃАрЃў рЃњрЃљрЃЏрЃЮрЃгрЃЋрЃћрЃЋрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљрЃерЃў.

-   рЃўрЃДрЃћрЃюрЃћрЃЉрЃА HMAC-рЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЏрЃўрЃћрЃа рЃЏрЃЮрЃгрЃЮрЃЊрЃћрЃЉрЃБрЃџрЃў рЃАрЃљрЃўрЃЊрЃБрЃЏрЃџрЃЮ рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃўрЃЌ (key.txt рЃцрЃљрЃўрЃџрЃерЃў рЃЕрЃљрЃгрЃћрЃарЃўрЃџрЃў рЃљрЃю рЃАрЃЎрЃарЃўрЃърЃбрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃАрЃљрЃА рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃА рЃ«рЃљрЃќрЃерЃў рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџрЃў рЃАрЃўрЃбрЃДрЃЋрЃљ)
-   рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЦрЃЏрЃюрЃўрЃА рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃцрЃљрЃўрЃџрЃА:
    -   `hashed_<рЃцрЃљрЃўрЃџрЃўрЃА_рЃАрЃљрЃ«рЃћрЃџрЃў>` Рђћ рЃљрЃюрЃЮрЃюрЃўрЃЏрЃўрЃќрЃћрЃЉрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў
    -   `map_<рЃцрЃљрЃўрЃџрЃўрЃА_рЃАрЃљрЃ«рЃћрЃџрЃў>` Рђћ рЃЮрЃарЃўрЃњрЃўрЃюрЃљрЃџрЃў рЃЊрЃљ рЃ░рЃћрЃерЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃбрЃЮрЃарЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃЮрЃЉрЃљ (рЃћрЃА рЃцрЃљрЃўрЃџрЃў рЃарЃЕрЃћрЃЉрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЏрЃцрЃџрЃЮрЃЉрЃћрЃџрЃЌрЃљрЃю)
-   рЃАрЃЎрЃарЃўрЃърЃбрЃўрЃА рЃЏрЃўрЃћрЃа рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃБрЃџрЃў рЃДрЃЋрЃћрЃџрЃљ рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ рЃўрЃгрЃћрЃарЃћрЃЉрЃљ рЃфрЃљрЃџрЃЎрЃћ `file_processor.log` рЃцрЃљрЃўрЃџрЃерЃў

## рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ

### 1. рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃњрЃљрЃЊрЃЏрЃЮрЃгрЃћрЃарЃљ

``` bash
git clone https://github.com/welwala206/georgian-id-hasher.git
cd georgian-id-hasher
```

рЃљрЃАрЃћрЃЋрЃћ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃЊрЃљрЃљрЃарЃЦрЃўрЃЋрЃћрЃЉрЃБрЃџ рЃцрЃљрЃўрЃџрЃљрЃЊ рЃњрЃљрЃЊрЃЏрЃЮрЃгрЃћрЃарЃљ. рЃљрЃЏрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА, рЃарЃћрЃърЃЮрЃќрЃўрЃбрЃЮрЃарЃўрЃА рЃќрЃћрЃЊрЃљ рЃЏрЃљрЃарЃ»рЃЋрЃћрЃюрЃљ рЃЎрЃБрЃЌрЃ«рЃћрЃерЃў рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃљрЃгрЃЎрЃљрЃърЃЮрЃЌ рЃЏрЃгрЃЋрЃљрЃюрЃћ рЃдрЃўрЃџрЃљрЃЎрЃА **Code** рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃЕрЃћрЃюрЃўрЃџ рЃЏрЃћрЃюрЃўрЃБрЃерЃў рЃЊрЃљрЃљрЃгрЃДрЃљрЃърЃЮрЃЌ **Download ZIP**.

### 2. рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃњрЃљрЃЏрЃљрЃарЃЌрЃЋрЃљ рЃЊрЃљ рЃАрЃљрЃГрЃўрЃарЃЮ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃўрЃА рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ

рЃАрЃЎрЃарЃўрЃърЃбрЃў рЃЊрЃљрЃгрЃћрЃарЃўрЃџрЃўрЃљ Python-рЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџ рЃћрЃюрЃљрЃќрЃћ, рЃљрЃЏрЃўрЃбрЃЮрЃЏ рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ Python 3 рЃБрЃЎрЃЋрЃћ рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃќрЃћ. рЃљрЃАрЃћрЃЋрЃћ, рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА Python-рЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃА, рЃарЃЮрЃЏрЃћрЃџрЃЌрЃљ рЃЕрЃљрЃЏрЃЮрЃюрЃљрЃЌрЃЋрЃљрЃџрЃўрЃф рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџрЃўрЃљ рЃцрЃљрЃўрЃџрЃерЃў requirements.txt. рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЉрЃљрЃбрЃБрЃарЃљрЃЊ рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА, рЃерЃћрЃљрЃАрЃарЃБрЃџрЃћрЃЌ рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ:

``` bash
pip install -r requirements.txt
```

> Windows-рЃќрЃћ рЃљрЃАрЃћрЃЋрЃћ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ:

## рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ

### рЃЎрЃЮрЃЊрЃБрЃарЃў рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃўрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ

рЃърЃўрЃарЃљрЃЊрЃў рЃюрЃЮрЃЏрЃарЃћрЃЉрЃўрЃА рЃАрЃарЃБрЃџрЃљрЃЊ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃ░рЃћрЃерЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃўрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ:

#### рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў 1: рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃўрЃЌ

``` bash
python main.py <рЃцрЃљрЃўрЃџрЃўрЃА_рЃљрЃю_рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА_рЃЏрЃЊрЃћрЃЉрЃљрЃарЃћрЃЮрЃЉрЃљ> --key "рЃЌрЃЦрЃЋрЃћрЃюрЃў_рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃў"
```

#### рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў 2: рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃўрЃА рЃцрЃљрЃўрЃџрЃўрЃЌ

``` bash
python main.py <рЃцрЃљрЃўрЃџрЃўрЃА_рЃљрЃю_рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА_рЃЏрЃЊрЃћрЃЉрЃљрЃарЃћрЃЮрЃЉрЃљ> --key-file key.txt
```

> рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃўрЃА рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃћрЃЉрЃџрЃЮрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃърЃўрЃарЃљрЃЊрЃў рЃюрЃЮрЃЏрЃарЃўрЃА рЃ░рЃћрЃерЃўрЃарЃћрЃЉрЃљ.

# ENGLISH VERSION

A lightweight Python utility to anonymize Georgian citizen IDs using **HMAC-SHA3-512**. This tool scans `.csv` and `.xlsx` files (or folders), detects 11-digit IDs, hashes them with a secret key, and generates anonymized files along with original-to-hashed mapping files.

## Use Case

This script is designed for **secure de-identification** of datasets containing personal identifiers (such as Georgian national IDs), ideal for:
- Health data research
- Government data processing
- Ethical data sharing

## Features

- Automatically detects and hashes **11-digit personal IDs**
- Supports **individual files** and **batch processing in folders**
- Handles both **CSV** and **Excel (.xlsx)** formats
- Uses **HMAC-SHA3-512** with user-supplied secret key
- Generates:
  - `hashed_<filename>` РђЊ anonymized dataset
  - `map_<filename>` РђЊ mapping of original Рєњ hashed IDs
- Logs all operations to `file_processor.log`

## Usage

### Providing the HMAC Key

The script supports two options for key input:
- `--key`: Pass key directly via command line
- `--key-file`: Path to a `.txt` file containing the key

### Examples

#### Process a single file:
```bash
python main.py ./data/file.xlsx --key "my_secret_key"
```

#### Process all `.csv` and `.xlsx` files in a folder:
```bash
python main.py ./data_folder --key-file key.txt
```

> If no key is provided, an empty string will be used (not secure).

## Output Files

For each input file, the script will generate:
- `hashed_<filename>.csv` or `.xlsx`: anonymized version
- `map_<filename>.csv` or `.xlsx`: original Рєњ hashed ID mapping
- `file_processor.log`: log of all actions

All files are saved in the same directory as the original.
