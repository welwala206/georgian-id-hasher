For install dependency:
pip install -r requirements.txt
or
py -m pip install -r requirements.txt

For run the program:
py main.py <path_to_file> - to process a file
py main.py <path_to_folder> - to process a folder
or
python main.py <path_to_file> - to process a file
python main.py <path_to_folder> - to process a folder

You can add parameter -k, --key for add some value to id before hashing for more security.
